# 📖 راهنمای استفاده از پنل مدیریت

## 🔐 ورود به پنل

1. ابتدا وارد سایت شوید
2. از منوی کاربری گزینه **Admin Panel** را انتخاب کنید
3. یا مستقیماً به آدرس زیر بروید:
   ```
   https://yoursite.com/Admin/Dashboard
   ```

## 👤 حساب مدیر پیش‌فرض

اطلاعات ورود به حساب مدیر که در دیتابیس seed شده:

```
نام کاربری: admin
رمز عبور: admin123
ایمیل: Admin@example.com
```

⚠️ **هشدار امنیتی**: حتماً بعد از اولین ورود، رمز عبور را تغییر دهید!

## 📊 بخش‌های پنل

### 1️⃣ داشبورد
اولین صفحه‌ای که بعد از ورود می‌بینید:
- آمار کلی سیستم (کاربران، پست‌ها، بازدیدها)
- آخرین کاربران ثبت‌شده
- آخرین پست‌های منتشر شده
- آخرین فعالیت‌های سیستم

### 2️⃣ مدیریت کاربران (`/Admin/Users`)
- **مشاهده لیست**: تمام کاربران با اطلاعات کامل
- **جستجو**: جستجو بر اساس نام، ایمیل، شماره تماس
- **جزئیات**: مشاهده اطلاعات کامل هر کاربر
- **عملیات**:
  - مسدود/فعال کردن حساب
  - حذف کاربر
  - مشاهده وضعیت تایید ایمیل و شماره

#### نحوه استفاده:
1. روی هر ردیف، آیکون‌های عملیات را می‌بینید
2. آیکون چشم 👁️ = مشاهده جزئیات
3. آیکون سپر 🛡️ = مسدود/فعال کردن
4. آیکون سطل زباله 🗑️ = حذف کاربر

### 3️⃣ مدیریت پست‌ها (`/Admin/Posts`)
- **مشاهده لیست**: تمام پست‌ها با وضعیت انتشار
- **ایجاد پست جدید**: دکمه سبز رنگ بالای صفحه
- **عملیات**:
  - ویرایش پست
  - انتشار/عدم انتشار
  - حذف پست
  - مشاهده تعداد بازدید

#### نحوه استفاده:
1. برای ایجاد پست جدید، دکمه **ایجاد پست جدید** را بزنید
2. هر کارت پست دارای دکمه‌های عملیات است
3. پست‌های منتشر شده با برچسب سبز، پیش‌نویس‌ها با زرد نمایش داده می‌شوند

### 4️⃣ فعالیت‌های اخیر (`/Admin/Activities`)
- **مشاهده Timeline**: تمام فعالیت‌های سیستم
- **اولویت‌بندی**: 
  - 🔴 اولویت بالا (قرمز)
  - 🟡 اولویت متوسط (زرد)
  - 🟢 اولویت پایین (سبز)
- **عملیات**:
  - حذف فعالیت منفرد
  - پاک کردن تمام فعالیت‌ها

### 5️⃣ تنظیمات (`/Admin/Settings`)
- **مشاهده تنظیمات**: تمام تنظیمات سیستم
- **ویرایش**: فقط تنظیمات قابل تغییر
- **انواع داده**:
  - متن (DataString)
  - عدد (DataInt)
  - بولین (DataBool)

⚠️ تنظیمات قفل شده (با آیکون قفل 🔒) قابل تغییر نیستند

## 🎨 راهنمای رابط کاربری

### رنگ‌ها
- **بنفش/صورتی**: عناصر اصلی و دکمه‌های اولیه
- **آبی**: اطلاعات و مشاهده
- **سبز**: موفقیت، فعال، تایید شده
- **زرد**: هشدار، تغییر وضعیت، پیش‌نویس
- **قرمز**: خطر، حذف، مسدود

### آیکون‌ها
- 🏠 خانه: داشبورد اصلی
- 👥 کاربران: مدیریت کاربران
- 📰 پست‌ها: مدیریت محتوا
- 📊 فعالیت‌ها: لاگ سیستم
- ⚙️ تنظیمات: پیکربندی
- 🌐 بازگشت به سایت
- 🚪 خروج

## 📱 نسخه موبایل

در دستگاه‌های موبایل:

2. کلیک روی آن برای باز کردن Sidebar
3. کلیک روی پس‌زمینه تیره برای بستن منو

## 🔒 نکات امنیتی

1. **رمز عبور قوی**: حتماً از رمز عبور قوی استفاده کنید
2. **تغییر رمز پیش‌فرض**: رمز `admin123` را فوراً تغییر دهید
3. **Logout**: همیشه بعد از کار، از سیستم خارج شوید
4. **دسترسی محدود**: فقط به افراد مورد اعتماد دسترسی مدیر بدهید

## 🐛 رفع مشکلات

### دسترسی محدود
اگر پیام "دسترسی محدود" دریافت کردید:
- مطمئن شوید با حساب مدیر وارد شده‌اید
- از صفحه خارج شده و دوباره وارد شوید
- Cache مرورگر را پاک کنید

### صفحه سفید
اگر صفحه سفید دیدید:
- صفحه را Refresh کنید
- Console مرورگر را بررسی کنید (F12)
- مطمئن شوید JavaScript فعال است

### خطای دیتابیس
اگر خطای دیتابیس دریافت کردید:
- مطمئن شوید دیتابیس در دسترس است
- Migration‌ها را اجرا کنید
- Connection String را بررسی کنید

# 🎨 پنل مدیریت حرفه‌ای

یک پنل مدیریت مدرن و حرفه‌ای با دیزاین Liquid Glass برای مدیریت کامل سایت

## ✨ ویژگی‌ها

### 🎯 داشبورد
- نمایش آمار کلی سیستم (کاربران، پست‌ها، بازدیدها، فعالیت‌ها)
- نمایش آخرین کاربران ثبت‌شده
- نمایش آخرین پست‌های منتشر شده
- نمایش آخرین فعالیت‌های سیستم
- طراحی ریسپانسیو و زیبا

### 👥 مدیریت کاربران
- مشاهده لیست تمام کاربران
- جستجو در میان کاربران
- مشاهده جزئیات کامل هر کاربر
- مسدود/فعال کردن کاربران
- حذف کاربران
- نمایش وضعیت تایید ایمیل و شماره تماس
- آمار کاربران (فعال، مسدود، امروز)

### 📰 مدیریت پست‌ها
- مشاهده لیست تمام پست‌ها
- ایجاد پست جدید
- ویرایش پست‌ها
- حذف پست‌ها
- انتشار/عدم انتشار پست‌ها
- نمایش تعداد بازدید هر پست
- آمار پست‌ها (کل، منتشر شده، پیش‌نویس، بازدید)

### 📊 مدیریت فعالیت‌ها
- مشاهده تمام فعالیت‌های سیستم
- نمایش اولویت فعالیت‌ها (بالا، متوسط، پایین)
- حذف فعالیت‌های منفرد
- پاک کردن تمام فعالیت‌ها
- نمایش Timeline فعالیت‌ها

### ⚙️ تنظیمات سایت
- مشاهده تمام تنظیمات سیستم
- ویرایش تنظیمات قابل تغییر
- قفل تنظیمات حساس
- نمایش انواع داده (متن، عدد، بولین)

## 🎨 طراحی

### Liquid Glass Effect
- افکت شیشه‌ای مایع با انیمیشن
- Backdrop blur برای جلوه گلس
- گرادیانت‌های رنگی زیبا
- انیمیشن‌های نرم و روان

### رنگ‌بندی
- **اصلی**: بنفش تا صورتی (#8b5cf6 → #ec4899)
- **پس‌زمینه**: تیره با گرادیانت (#0a0a0a → slate-900)
- **Accent**: رنگ‌های مختلف برای وضعیت‌ها

### آیکون‌ها
- Font Awesome 6.4.0
- آیکون‌های واضح و معنادار
- استفاده مناسب در تمام بخش‌ها

### فونت
- **فارسی**: Vazirmatn
- وزن‌های مختلف: 300 تا 800
- خوانایی عالی

## 📱 Responsive Design
- موبایل: منوی کشویی
- تبلت: نمایش بهینه
- دسکتاپ: Sidebar ثابت
- تمام صفحات کاملاً ریسپانسیو

## 🔐 امنیت
- Authentication با ASP.NET Identity
- Authorization برای محدود کردن دسترسی
- صفحه Access Denied اختصاصی
- CSRF Protection

## 🔧 تکنولوژی‌ها

- **Framework**: ASP.NET Core 8.0
- **CSS Framework**: Tailwind CSS 3.x
- **Icons**: Font Awesome 6.4.0
- **Database**: Entity Framework Core
- **Authentication**: ASP.NET Identity

## 📝 نکات مهم

1. **احراز هویت**: کاربران باید وارد سیستم شده باشند
2. **مجوز دسترسی**: در آینده می‌توان Role-Based Access Control اضافه کرد
3. **واکنش‌گرایی**: تمام صفحات با موبایل سازگار هستند
4. **کارایی**: استفاده از Lazy Loading برای بهینه‌سازی
